# 🚀 시작하기 - JAKUP 작업관리 시스템

환영합니다! 이 가이드는 프로젝트를 **빠르게 시작**하기 위한 간단한 안내입니다.

---

## 🎯 선택하세요

### 1️⃣ **로컬에서 개발하기** (개발자용)
컴퓨터에서 프로젝트를 실행하고 수정하려면:

👉 **[개발 시작 가이드 (README.md)](./README.md)**

**필요한 것**: Node.js, PostgreSQL, 약 30분

---

### 2️⃣ **웹에 배포하기** (초보자 친화적) ⭐ 추천
인터넷에서 바로 사용 가능하게 만들려면:

👉 **[Railway 빠른 배포 (30분)](./QUICK_START_RAILWAY.md)**

**필요한 것**: GitHub 계정, 신용카드 (무료 크레딧), 약 30-60분  
**비용**: 월 $0-5 (무료 크레딧으로 충분)

**장점**:
- ✅ 클릭 몇 번으로 배포
- ✅ 자동 SSL (HTTPS)
- ✅ 자동 백업
- ✅ 전 세계 어디서나 접속 가능

---

### 3️⃣ **상세한 배포 가이드** (여러 옵션)
다양한 배포 방법 (Docker, AWS, VPS 등):

👉 **[전체 배포 가이드 (DEPLOYMENT.md)](./DEPLOYMENT.md)**
